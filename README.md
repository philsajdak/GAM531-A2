# GAM531-A2
GAM531 Assignment 2: Vector & Matrix Operations in C#

## Library Used
OpenTK.Mathematics

## Operations Implemented

### Vector Operations
- Addition
- Subtraction  
- Dot Product
- Cross Product

### Matrix Operations
- Identity Matrix
- Scaling Matrix
- Rotation Matrix (Y-axis)
- Matrix Multiplication
- Vector Transformation
